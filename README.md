Brings modern text editing to the Minecraft Beta 1.7.3 chat. No more being stuck at the end of your message.

## Features
- **Arrow keys** to move the cursor
- **Shift + arrows** to select text
- **Home / End** to jump to start or end
- **Ctrl + X** to cut
- **Ctrl + C** to copy
- **Ctrl + V** to paste
- **Ctrl + A** to select all
- **Click** to place cursor, **click & drag** to select
- **Backspace / Delete** respect selections
- Typing **replaces** selected text instantly

## Installation

[![Download on Modrinth](https://img.shields.io/badge/Modrinth-Download-brightgreen?style=for-the-badge&logo=modrinth)](https://modrinth.com/mod/better-beta-chat)

### MultiMC / Prism Launcher
1. Right-click your Beta 1.7.3 instance and select **Edit Instance**
2. Go to the **Loader Mods** or **jar mods** tab
3. Click **Add to Minecraft.jar** and select the downloaded `.zip`
4. Launch the game

### Manual
1. Locate your `minecraft.jar` (`.minecraft/bin/minecraft.jar`)
2. Open it with any archive tool (e.g. 7-Zip or WinRAR)
3. Delete the **META-INF** folder inside the jar
4. Drag and drop the mod files into the jar
5. Save and launch the game
